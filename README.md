# PRESENT NOW 

![alt text](https://github.com/Rahul-Agarwal-07/Present-Now/blob/master/app/present-now-logo.png)

## About Application

- Present Now is a presentation management application.
- It helps to maintain presentation without any physical record keepings.

## Objectives

- Centralizes all the presentation data at one place.
- Provides transparency to all the users.

## Applications
- Mainly to be used in colleges
- Schools

## How to Use?

- Login with your Google id
- Two Options :
    - Create - Create new instance, to be used by admin of the presentation
    - Join - Enter code to join specific ppt instance
      
- Group Creation & Topic selection :
    - Can form groups of user choices
    - Topic can be selected
      
- Groups and Topics Tab :
  
    - Shows information about group and its members
    - Shows topics selected and the time when it was selected.
